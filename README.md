## 🚀 How to Use

1. **Clone the repository**:

   ```bash
   git clone https://github.com/HungDongne/LeetCode.git
   cd LeetCode
   ```

2. **Run Python solutions**:

   ```bash
   python [problem_number].py
   ```

3. **Compile and run C++ solutions**:
   ```bash
   g++ -o solution [problem_number].cpp
   ./solution
   ```
